üß© 2Ô∏è‚É£ POINTERS & ARRAYS
--------------------------------------------

Pointer basics:
int a=10; int *p=&a; printf("%d", *p); // 10

Pointer arithmetic:
p+1 ‚Üí next element
p2 - p1 ‚Üí element gap
(char*)p2 - (char*)p1 ‚Üí byte gap

Example:
int arr[]={10,20,30,40,50};
int *p1=arr, *p2=arr+4;
printf("%d", p2-p1); //4
printf("%d",(char*)p2-(char*)p1); //16 bytes

Pre/Post increment:
*ptr++ ‚Üí use then move
*++ptr ‚Üí move then use
(*ptr)++ ‚Üí increment value

Array-pointer equivalence:
arr[i] == *(arr+i) == i[arr]
